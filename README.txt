INTRODUCTION
------------

The MOS registration module displays the entire set of data atrributes
that are required to be submitted by end user before using 
MeetOnSnap services 

 * For a full description of the module, visit the project page:
   http://meetonsnap.com

 * To submit bug reports and feature suggestions, or to track changes:
   http://meetonsnap.com/admin_menu


REQUIREMENTS
------------

This module requires the following modules:

 * Smart Forms (http://doptiq.com/smart-forms/)
 * JQuery (https://jquery.com/)
 * HTML5 (https://www.w3.org/TR/html5/)
 * CSS3 (http://www.w3schools.com/css/css3_intro.asp)
 * PHP (http://php.net/)
 * CSS Media Query (http://www.w3schools.com/cssref/css3_pr_mediaquery.asp)

INSTALLATION
------------

 * Install the registration module by simply uncompressing the zip file
 * Open your favourite web browser and view index.php
 * Gravatar images are saved in uploads folder in the project directory

CONFIGURATION
-------------

 * Configure the web browser to enable javascript
 * Enable support to access browser cookies


TROUBLESHOOTING
---------------

* If the registration page does not load, check the following:

   - Browser supports for HTMl5, CSS3 & Javascript
